Qui suis-je ?

Depuis longtemps passionné d'informatique, je possède des connaissances dans les domaines du web (conception et réalisation de sites internet), de la programmation (conception et réalisation d'applications à destination des entreprises ou collectivités dans divers langages de programmation) et de l'administration système (notamment sous GNU/Linux).

Dans le domaine de la programmation, je développe et utilise uniquement des logiciels libres (sous licence GPL, Mozilla, BSD, ...) ; j'utilise personnellement les systèmes d'exploitation GNU/Linux Fedora Core et Ubuntu.

Dans le domaine du développement web, je milite pour une accessibilité à tous, et en conséquence respecte les standards établis par le W3C ((X)HTML, CSS, WAI/WCAG, XML, XSL, ...) ; les réalisations web sont testées afin de fonctionner de façon similaire sur les différents navigateurs web que l'on peut rencontrer (Mozila, Firefox, Konqueror, Internet Explorer, Opera, ...), chacun ayant ses propres spécificités.

Copyright et restrictions

Les éléments contenus sur les sites de la partie "Sites Web" appartiennent à leurs propriétaires respectifs et peuvent être soumis à certaines restrictions. Tout le reste est entièrement libre de droits.

Curriculum Vitae (Diverses possibilités d'affichage d'un curriculum vitae stocké dans un fichier XML)

Rendu par XSL via PHP Au vu des problèmes possibles de différences de rendus entre navigateurs, cette approche est plus universelle. Cette fois la transformation XSL est appliquée côté serveur grâce à PHP. Cette méthode fonctionnera avec tous les butineurs.

Rendu par XSL côté navigateur Les navigateurs web possèdent un moteur de rendu XSL interne. En associant une feuille de style XSL au fichier XML, il est possible d'en effectuer le rendu directement. les butineurs non compatibles afficheront le contenu du fichier XML en ligne.

Rendu au format PDF Le format PDF est un format à préférer dans les emails, car il vous assure que sa mise en page sera la même chez vous et chez votre destinataire ; ce qui n'est pas le cas d'un fichier html et encore moins d'un fichier issu d'un traitement de texte.

Réalisations PHP
Vidéothèque Réalisée en PHP/MySQL, cette application vise la gestion d'une vidéothèque. Il est possible de gérer en ligne les enregistrements, de faire des réservations (grâce aux identifiants de l'utilisateur) ou encore de gérer des sorties (ou prêts). Cette application a récemment effectué une migration vers php5, seules les fonctionnalités de base (enregistrement, recherches, ...) sont disponibles, les autres sont en cours de réécriture. PHP5 - MySQL
Calendrier Artistes Calendrier pour artistes réalisé en PHP/MySQL, le calendrier utilisé pour la base de ce projet est un projet open source nommé PHPiCalendar, sur la demande d'une association artistique afin de gérer les emplois du temps de ses musiciens, groupes et salles de répétitions. PHP - MySQL
