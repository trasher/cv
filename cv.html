Qui suis-je ?

Depuis longtemps passionné d'informatique, je possède des connaissances dans les domaines du web (conception et réalisation de sites internet), de la programmation (conception et réalisation d'applications à destination des entreprises ou collectivités dans divers langages de programmation) et de l'administration système (notamment sous GNU/Linux).

Dans le domaine de la programmation, je développe et utilise uniquement des logiciels libres (sous licence GPL, Mozilla, BSD, ...) ; j'utilise personnellement les systèmes d'exploitation GNU/Linux Fedora Core et Ubuntu.

Dans le domaine du développement web, je "milite" pour une accessibilité à tous, et en conséquence respecte les standards établis par le W3C (XHTML, CSS, WAI/WCAG, XML, XSL, ...) ; mes réalisations web sont testées afin de fonctionner de façon similaire sur les différents navigateurs web que l'on peut rencontrer (Mozila, Firefox, Konqueror, Internet Explorer, Opera, ...), chacun ayant ses propres spécificités.

Les éléments contenus sur les sites de la partie "Sites Web" appartiennent à leurs propriétaires respectifs et peuvent être soumis à certaines restrictions. Tout le reste est entièrement libre de droits.

Curriculum Vitae (Diverses possibilités d'affichage d'un curriculum vitae stocké dans un fichier XML)

Rendu par XSL via PHP (XML - PHP5 - XSLT) Au vu des problèmes possibles de différences de rendus entre navigateurs, cette approche est plus universelle. La transformation XSL est appliquée côté serveur grâce à PHP. Cette méthode fonctionne avec tous les butineurs

Rendu par XSL côté navigateur (XML - XSLT - CSS) Les navigateurs web possèdent un moteur de rendu XSL interne. En associant une feuille de style XSL au fichier XML, il est possible d'en effectuer le rendu directement.. Les butineurs non compatibles afficheront le contenu du fichier XML en ligne.

Rendu au format PDF (XML - PHP5 - FPDF) Le format PDF est un format à préférer dans les emails, car il vous assure que sa mise en page sera la même chez vous et chez votre destinataire ; ce qui n'est pas le cas d'un fichier html et encore moins d'un fichier issu d'un traitement de texte.

Rendu via Flash (XML - Flash) Chargement du fichier XML dans une animation Flash.

Réalisations PHP

Gestion de Vidéothèque (PHP5 - MySQL) Réalisée en PHP/MySQL, cette application vise la gestion d'une vidéothèque. Il est possible de gérer en ligne les enregistrements, de faire des réservations (grâce aux identifiants de l'utilisateur) ou encore de gérer des sorties (ou prêts). Cette application a récemment effectué une migration vers php5, seules les fonctionnalités de base (enregistrement, recherches, ...) sont disponibles, les autres sont en cours de réécriture.

Calendrier Artistes (PHP - MySQL) Calendrier pour artistes réalisé en PHP/MySQL, le calendrier utilisé pour la base de ce projet est un projet open source nommé PHPiCalendar, sur la demande d'une association artistique afin de gérer les emplois du temps de ses musiciens, groupes et salles de répétitions.

Réalisations Java (Retrouvez le détail des principales conceptions logicielles en cliquant sur le titre)

Gestion de cantine (Java) Conception et réalisation d'un logiciel de gestion de cantine scolaire sous licence GPL pour une mairie. Gestion des familles et élèves, gestion des jours normaux de présence, planning (absences prévues, non prévues, présences imprévues, présences prépayées, facturation en fonction du planning, gestion des congés et jours fériés, impressions diverses, ...

Reminder eGroupWare (Java) Un reminder basé sur eGroupWare, un outil de travail collaboratif en PHP. eGroupWare propose par défaut un rappel par email pour vos rendez-vous, mais c'est une solution qui trouve vite ses limites (perte de connexion internet, logiciel email fermé, pas d'accès internet !, ...). Le reminder se lance dans votre barre des tâches (testé sous Windows XP/2K et KDE/Linux). Les sources sont incluses dans le package d'installation.

BibliothèK (Java) Gestion de bibliothèque en java. Sur la demande d'un ami et mon besoin personnel. Les sources sont incluses dans le package d'installation, ainsi que les instructions nécessaires à l'installation et à l'utilisation du programme.

Sites Web

Hardware & Communication (XML - PHP5 - XSLT - MySQL) Conception et réalisation du site web de la société Hardware & Communication ; mises à jours par fichiers XML, rendu XSLT/CSS, programmation PHP. Partie clients accessible via login pour l'accès aux statistiques par AwStats ainsi qu'aux versions en cours de travail. Site validé XHTML 1.0 et CSS.

Mutuelle Choralis (PHP - MySQL) Conception et réalisation du site web de la mutuelle Choralis, sise à Douai. Menus dynamiques, animations flash, système de devis en ligne en PHP, fil de news.

Association Cénacle de Douayeul (XML - PHP5) Conception et réalisation du site web du Cénacle de Douayeul, association de poésie du douaisis. Mise à jour de parties du site par fichiersXML, partie administration, page contact et écriture en ligne dynamiques.

Association Roulards Sans Frontières (PHP5 - XML) Conception et réalisation du site web de Roulards-Sans-Frontières, association pour l'accessibilité aux voyages pour les personnes handicapées. Mise à jour des destinations, liens et contenus divers par fichiers XML, respect des normes XHTML 1.0, CSS et WAI triple A.

Complexe Saintes Vegas (PHP5 - XML) Conception et réalisation du site web du complexe Saintes Vegas (17). Validé XHTML et CSS.

Dylath-Leen (XML - PHP) Mise en place d'une galerie pour afficher les photos des concerts du groupe. Le site est valide XHTML et CSS, il possède une déclaration Opquast.

Cantine (Xoops) Mise en place d'un site pour mon projet de gestion de cantines scolaires. Site validé XHTML et CSS.

X-TnD (PHP5 - XML - XSL ) Conception et réalisation du site web X-TnD (sur lequel sont hébergées ces pages). Ce site est en constante évolution, il est valide XHTML 1.1, CSS 2.0 et WAI AA. Une déclaration Opquast afin de mener une étude qualitative du site a été effectuée.
